🌳 Ancestry Fan Chart Generator

Welcome! This application creates beautiful, interactive fan charts from your GEDCOM genealogy files.

Launch Application →

What You'll Need

  • A GEDCOM (.ged) file exported from your genealogy software (Ancestry, MyHeritage, FamilySearch, etc.)
  • A modern web browser

Quick Start

  1. Click the "Launch Application" button above
  2. Click "Upload GEDCOM File" and select your .ged file
  3. Select a center person from the dropdown menu
  4. Adjust settings as desired (generations, fonts, colors)
  5. Click "Generate Chart" to create your fan chart
  6. Click on any person to view their details
  7. Download your chart as SVG when ready

Features

  • ✨ Interactive Charts - Click any person to see details
  • 🎨 Customizable - Choose colors, fonts, and what information to display
  • 📊 Flexible - Show 1-8 generations of ancestors
  • 👨‍👩‍👧‍👦 Descendants - Optionally include descendants
  • 💾 Export - Download as SVG for printing or further editing
  • 🔒 Private - All processing happens in your browser - no data is uploaded

Exporting GEDCOM from Popular Services

Ancestry.com

  1. Go to your family tree
  2. Click "Trees" → "View Tree"
  3. Click the tree name → "Tree Settings"
  4. Select "Export tree" → "Export to GEDCOM"

MyHeritage

  1. Go to Family Tree → Manage tree
  2. Click "Download" → "Export to GEDCOM"

FamilySearch

  1. Sign in and go to Family Tree
  2. Click on a person → "Tools" → "Download"
  3. Select GEDCOM format

Tips for Best Results

  • Start with 4-5 generations for optimal readability
  • Use larger font sizes for printing
  • Try different color schemes to find your preference
  • Enable "Show Country" if you have international ancestry
  • Export as SVG to edit in vector graphics programs like Adobe Illustrator or Inkscape

For Developers

This is a pure client-side application built with vanilla JavaScript. To customize or extend:

  • gedcom-parser.js - Handles GEDCOM file parsing
  • fan-chart.js - Generates the SVG fan chart
  • app.js - Main application logic and UI
  • styles.css - Styling and layout

Privacy & Security

Your family data is precious. This application:

  • ✅ Runs entirely in your browser
  • ✅ Does not upload your GEDCOM file anywhere
  • ✅ Does not store any data on servers
  • ✅ Does not require an account or login
  • ✅ Can be used completely offline

Support

Having trouble? Check these common solutions:

  • Chart not generating? Make sure you've selected a center person
  • File won't upload? Ensure it's a valid .ged GEDCOM file
  • Text overlapping? Try reducing the number of generations or increasing font size
  • Performance issues? Large family trees may take time to process
